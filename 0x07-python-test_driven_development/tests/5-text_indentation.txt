=================================
How to use 5-text_indentation.py
=================================

This module defines a text-indentation function ``text_indentation(text)``.

USAGE
======

Texts is printed with two newlines after each of the characteers ``.``, ``?``,
 and ``:``

::

	>>> text_indentation = __import__('5-text_indentation').text_indentation
	>>> text_indentation("Antelope?")
	Antelope?
	<BLANKLINE>


 Spaces are not printed at the beginning of the line.

::
	
	>>> text_indentation("  Heyyo.")
	Heyyo.
	<BLANKLINE>

prints nothing if an empty string is passed

::
	>>> text_indentation("       ")

Spaces are not printed at the end of the line even if the argument passed has
 a space at the end

::

	>>> text_indentation("ALX study group:    ")
	ALX study group:
	<BLANKLINE>
